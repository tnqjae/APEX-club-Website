# 🧠 Apex

Apex Club Website
Apex는 인공지능, 개발, 보안에 관심 있는 학생들이 함께 학습하고 협업하는 대학생 동아리입니다.  
이 저장소는 Apex 동아리의 공식 웹사이트로, 동아리 소개, 지도 교수님, 팀원, 프로젝트, 갤러리 등의 정보를 제공합니다.

# 📁 Project Structure
src/
├── assets/            # 이미지 및 정적 파일
├── components/        # 재사용 가능한 Vue 컴포넌트
├── views/             # 페이지별 Vue 컴포넌트
├── router/            # 라우터 설정
├── App.vue            # 루트 컴포넌트
└── main.js            # 앱 진입점
